# Crypto Promotion Bot

多链加密货币推广机器人，支持ETH、BSC、SOL、BASE链的支付和推广系统。

## 功能特点

### 1. 多链支付系统
- 支持ETH、BSC、SOL、BASE链支付
- 自动验证交易
- 支付超时处理
- 多币种余额管理

### 2. 置顶推广系统
- 多级置顶位置
- 智能排队系统
- 动态价格调整
- 自动续费提醒

### 3. 邀请返佣系统
- 三级返佣架构
- 实时返佣计算
- 多链提现支持
- 收益统计分析

### 4. 实时监控系统
- 交易监控
- 系统性能监控
- 异常报警
- 数据统计报告
